Operatoren

Man kann für eine Anweisung auch mehrere Bedingungen in nur einer if-Anweisung formulieren. Dabei werden die Vergleichsoperationen, die Bedingungen mittels Logischem Operator verbunden. Sollen zwei Bedingungen true sein, wird das logische UND verwendet, mit && als Symbol. Dieses erwartet auf seinen beiden Seiten einen true-Wert, um selber ein true zurückgeben zu können. Ist auch nur auf einer der beiden Seiten oder auf beiden ein false-Wert, wird auch das logische UND false ergeben
